📊 E-commerce Data Analysis (Kaggle + Python + Power BI)
🚀 Sobre o projeto

Este projeto tem como objetivo realizar uma análise completa de um dataset de e-commerce utilizando dados públicos do Kaggle (Olist), passando por etapas de tratamento, transformação e visualização de dados.

O foco principal foi entender o comportamento de vendas, desempenho logístico e indicadores de entrega.

🧠 Etapas do projeto
1️⃣ Coleta de dados
Dataset público utilizado: Brazilian E-Commerce Public Dataset by Olist (Kaggle)
2️⃣ Tratamento de dados (Python)
Limpeza e preparação dos dados com Pandas
União de múltiplas tabelas do dataset
Criação de base consolidada para análise
Exportação dos dados tratados para uso no Power BI
3️⃣ Análise e visualização (Power BI)

Criação de dashboard interativo com os seguintes indicadores:

📊 Principais análises realizadas:

Faturamento mensal
Ranking de receita por categoria de produtos
Status de entrega (atrasado vs no prazo)
Produtos com maior índice de atraso
Tempo médio de entrega
📈 Principais insights
Identificação das categorias de produtos com maior faturamento
Análise de desempenho logístico e atrasos nas entregas
Visão geral do comportamento de vendas ao longo do tempo
Monitoramento de eficiência no processo de entrega
